## Global Service - Profile: Get Profiles

URL: https://global-profile-service.game-social.epicgames.com/profiles \
Method: PUT \
Auth Required: Yes (No Perm required)

```json
{
  "namespace": "",
  "accountIds": []
}
```

## Parameters

`namespace`: e.g. `Fortnite` \
`accountIds`: the account ids (up to 100)
