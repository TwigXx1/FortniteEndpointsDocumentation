## Fortnite Website - Video Clips

URL: https://fortnite.com/:language/api/video-clips \
Method: GET \
Auth Required: Yes

<br/>

## Fortnite Website - Video Clips Post

URL: https://fortnite.com/:language/api/video-clips \
Method: POST \
Auth Required: Yes

Payload:

```json
[
  {
    "key": "scene24",
    "val": "yes"
  },
  {
    "key": "user_color",
    "val": "blue"
  },
  {
    "key": "user_soundtrack",
    "val": "BunkerJam"
  }
]
```

The values can be gotten from the get request if you're eligible
